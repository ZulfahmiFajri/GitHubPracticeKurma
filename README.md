- Perkalian (`mul`)

## Cara Menggunakan Program

### Clone and Open Project

```bash
git clone https://github.com/ZulfahmiFajri/GitHubPracticeKurma.git
cd GitHubPracticeKurma
```

### Compile

```bash
gcc main.c add.c sub.c mul.c div.c -o kalkulator
```

### Run

```bash
kalkulator.exe
```
